Flexbox Demonstrations

Now it's time to demonstrate spacing between elements. For this, I'll use my set of 16 boxes ... if the boxes are set to a percentage size, then flexbox sizes all of them to fit the container. If, on the other hand, a fixed width is chosen (in this case 120 pixels), then space appears (in grey below) where a fixed size contained element won't fit. The default method of handling the appearance of that extra space is given by the CSS declaration "justify-content: flex-start", which forces all elements inside the container to begin at the zero coordinate value of the primary ordering axis (in this case the horizontal axis), and leaves space at the opposite end of the primary ordering axis:
