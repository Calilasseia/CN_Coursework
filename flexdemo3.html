Flexbox Demonstrations

Now it's time to provide something resembling a real world example, namely, how to construct a clickable image gallery using flexbox. I'll reuse the images from my own gallery in the main pages and illustrate how flexbox brings a certain elegance and simplicity to this task. SPECIAL NOTE : Each of the items in the main flexbox in this demo, is itself another flexbox, which demonstrates that you can nest flexboxes if your design calls for this! Of course, this is probably overkill for this simple example, but illustrates the principle in action. Examination of the HTML and CSS files for this particular demo will prove illuminating.
